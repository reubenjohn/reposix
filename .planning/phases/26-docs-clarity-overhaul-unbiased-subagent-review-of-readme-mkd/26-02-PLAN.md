How to invoke it (from SKILL.md process):
1. Create a unique temp dir: REVIEW_DIR=$(mktemp -d /tmp/doc-clarity-review-XXXXXX)
2. Write the default clarity prompt to $REVIEW_DIR/_prompt.md
3. Copy the target file(s) to $REVIEW_DIR/
4. Run: claude -p "$(cat $REVIEW_DIR/_prompt.md)" $REVIEW_DIR/<file> 2>&1 | tee $REVIEW_DIR/_feedback.md
5. Read $REVIEW_DIR/_feedback.md for Verdict + friction points
6. Fix the original file based on feedback
7. Re-run the review on the fixed file to confirm CLEAR

The default clarity prompt asks: friction points, unanswered questions, over-explained, under-explained, missing references, and a CLEAR/NEEDS WORK/CONFUSING verdict.
