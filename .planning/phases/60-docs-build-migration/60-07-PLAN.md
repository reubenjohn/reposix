   - **mkdocs-strict RED** (any "Syntax error in text" OR mkdocs --strict warning): inspect the build log; fix the offending markdown / mkdocs.yml / mermaid block.
   - **mermaid-renders RED** (artifact missing OR svg_counts has zero OR console_errors non-empty): refresh artifacts via `mcp__playwright__*` walk per CLAUDE.md "Whole-nav-section playwright walks" rule. The walks MUST run cold (cache disabled) per the §0.1.b race condition.

   - **badges-resolve RED** (any URL 404 / wrong content-type): swap broken URLs for current shields.io equivalents; verify post-fix.

3. **Fix RED code rows** (Task 3):
   - **cargo-fmt-check RED**: `cargo fmt --all` to auto-fix; commit.
   - **cargo-clippy-warnings RED**: investigate origin (is it new in P60? is it transitive crate update?); in-phase fix OR file SURPRISES + waive + MIGRATE-03 carry-forward.

4. **Fix RED structure rows** (Task 4):
   - **cred-hygiene RED**: a real credential leak. STOP. Rotate the token at the issuing platform. Do NOT push.
   - **badges-resolve RED**: same as docs-build/badges-resolve (shared verifier).

5. **Re-run runner sweep + commit fixes** (Task 5): after fixes land, re-run `--cadence pre-push` and confirm every P60-touched row grades PASS (or PARTIAL/FAIL with documented MIGRATE-03 carry-forward). Commit all fixes in 1-N commits per the natural fix granularity.

**Pivot rules** (from the prompt):
- **>5 broken links**: fix the worst N, file backlog for rest, do NOT block phase close.
- **shields.io rate-limit transient (HTTP 429)**: mark badges-resolve PARTIAL; row stays WAIVED until next cadence; v0.12.1 carry-forward for retry-with-backoff.
- **GH Pages cache hasn't propagated post-Wave-F**: badges-resolve PARTIAL on QG-09 URL is acceptable; row is P2; doesn't block. SUMMARY documents the lag; Wave H verifier subagent grades on the runner output, not the propagation.
- **Mermaid playwright walk requires fresh-context browser**: per CLAUDE.md "Whole-nav-section playwright walks" rule, walks MUST run with cache disabled (fresh first-load semantics; the §0.1.b race condition only manifests on cold page-cache; warm reloads mask it). Use `mcp__playwright__browser_navigate` on a clean context.
- **Cargo clippy reports a NEW warning vs P59 close**: investigate. If the warning is genuinely new in P60's commits, fix. If from transitive crate update outside P60 scope, file SURPRISES + waive + MIGRATE-03 carry-forward.

**Operating-principle hooks**:
- **OP-1 close the feedback loop**: the runner fires the verifiers; the verifiers find RED; Wave G fixes RED; the next runner invocation confirms GREEN. Don't write fixes "blind"; let the runner verify.
- **OP-4 self-improving infrastructure**: each RED that Wave G surfaces is evidence that the gate is doing its job. Document in SURPRISES.md if the RED was a genuine regression caught by the new framework -- this is the framework earning its keep.
