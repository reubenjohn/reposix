The short-lived waiver pattern (P58 Wave A invented it; P59/P60 used it) keeps every cadence GREEN through the implementation window. Waivers expire 2026-05-15 (~18d from this commit at 2026-04-27); after Wave G dispatches the rubrics + persists artifacts, the runner re-grades to PASS and the waivers expire harmlessly.
