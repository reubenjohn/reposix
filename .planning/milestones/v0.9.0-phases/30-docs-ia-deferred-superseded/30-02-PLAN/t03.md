# Connect to GitHub

!!! note "Stub — this page points readers at existing reference + demo material. Post-launch, it grows into a step-by-step setup guide."

reposix ships a GitHub backend in `crates/reposix-github`. To connect a real GitHub project:

1. Export a PAT: `export GITHUB_TOKEN="$(gh auth token)"`.
2. Export the allowlist: `export REPOSIX_ALLOWED_ORIGINS='http://127.0.0.1:*,https://api.github.com'`.
3. See `scripts/demos/05-mount-real-github.sh` for a working one-shot example.

Full reference: see the [GitHub section in the simulator reference](../reference/simulator.md) and the demo scripts under `scripts/demos/`.
