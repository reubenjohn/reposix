FIX APPLIED (the B2 root cause, cleared)
----------------------------------------
The verifier read `SPACE="${REPOSIX_CONFLUENCE_SPACE:-TokenWorld}"`, which picked up
REPOSIX from the ambient .env (the READ-ONLY durable fixture space), then a
sanctioned-target `case` hard-FAILed `exit 1` because REPOSIX != TokenWorld — so the
row SELF-REJECTED under normal creds and could never run. Since this row CREATES +
MUTATES Confluence pages, TokenWorld is the ONLY sanctioned target for it. The block was
replaced with a teaching comment + a single line:

    export REPOSIX_CONFLUENCE_SPACE=TokenWorld

Pinning is strictly SAFER than rejecting: the smoke can now only ever mutate the
sanctioned space, whatever .env points at. Confirmed no dangling `$SPACE` remains
(grep: only the pin line + a comment reference).
