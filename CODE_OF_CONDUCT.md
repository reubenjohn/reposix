# Code of conduct

This project adopts the [Contributor Covenant Code of Conduct, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be patient, assume good faith, and focus on the technical substrate. The reposix project deliberately operates in a high-leverage agentic engineering space — we want contributors comfortable being honest about what works and what does not.

## Reporting concerns

If you experience or witness conduct in this project that violates the Covenant, contact:

- **Email:** reubenvjohn@gmail.com
- **GitHub Security Advisory:** for sensitive reports — <https://github.com/reubenjohn/reposix/security/advisories/new>

Reports will be acknowledged within 7 days. Reporters will not be retaliated against.

## Scope

This Covenant applies to all project spaces — issues, PRs, discussions, the docs site, related social channels — and applies to behavior toward maintainers and other contributors.

## Enforcement

Maintainers may take any action they deem appropriate, including warning, temporary ban, or permanent ban. Decisions are made transparently in private maintainer channels and recorded for accountability.
